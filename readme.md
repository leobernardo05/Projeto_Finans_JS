# 💰 Orçamento Pessoal

Aplicação web desenvolvida para realizar o controle de despesas pessoais diretamente no navegador, utilizando **JavaScript**, **HTML**, **Bootstrap** e **LocalStorage** como forma de persistência de dados.

O sistema permite cadastrar, consultar e filtrar despesas de forma simples, simulando um pequeno sistema de gestão financeira.

---

## 🚀 Funcionalidades

* 📌 Cadastro de despesas
* 🔎 Consulta e filtragem de despesas
* 💾 Armazenamento dos dados no **LocalStorage**
* 📊 Listagem dinâmica das despesas
* ⚠️ Validação de campos obrigatórios
* 🎨 Interface responsiva com **Bootstrap**
* 📦 Organização do código utilizando **Classes em JavaScript (ES6)**

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 4
* jQuery
* LocalStorage (Web Storage API)

## ⚙️ Como executar o projeto

1. Clone este repositório

```
git clone https://github.com/seu-usuario/orcamento-pessoal.git
```

2. Acesse a pasta do projeto

```
cd orcamento-pessoal
```

3. Abra o arquivo **index.html** no navegador.
